Welcome to the Project Monitoring System of SSEL

The Project Monitoring System of SSEL is one of the most advanced tools to monitor the projects that SSEL executes all over the India, the project monitoring system makes use of an android application deployed to all the supervisors and managers of SSEL, the data captured by them from the project is relayed over the web application where the data is formalized, analysed and put to right use.
